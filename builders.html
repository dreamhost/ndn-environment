Perl
This will download, configure, and build the perl version specified in the config file or command line.
Cpanm
This will install cpanm into the custom built perl.
EnvPAN
This will bootstrap OrePAN2 and inject the modules listed in your config file from cpan.

Note: It is best to add the envpan directory this builds to your project git repo, that way building from fresh checkouts of your environment will build with the same module versions.

Modules
Build all the modules listed in your config file, as well as their dependencies. Will ONLY use modules that have been added to envpan. If envpan is missing a dependency then this build will fail. There is an --auto_deps flag that will make envpan inject deps from cpan as it finds them, which is useful for initial bootstrapping.
Static
Will copy the static/* files to build/* so that they will be included in any package that is built.
ModPerl
Will build mod_perl against the system apache and your custom perl.
ImageMagick
Builds ImageMagick against the custom perl.
Package
Created a package for your environment.
CUSTOM
Builders are plugins, it is easy to add custom ones.
