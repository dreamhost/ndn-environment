Purge
Simply removes the 'build' directory so that the environment will need to be built completely from scratch again.
Init
Used to initialize a new environment. This will create the 'myenv' directory with a config file example and an empty 'source' directory.
Help
Used to get help. Without arguments it will show all available commands. With a command as an argument it will provide command specific help. The help for the 'build' command lists available builders.
Build
Used to build one or more components of the environment. If you request to build a component that depends on others they will be built as well.
AddModule
This will add the specified module to the environment. Note: This does not add the module to the config file, so building the environment fram scratch will not include the added module.
Run
This will run the custom-build perl binary (assuming it is built).
CUSTOM
Commands are plugins, it is easy to add custom ones.
